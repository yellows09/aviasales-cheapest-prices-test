# Aviasales Telegram Bot

Telegram-бот для поиска дешёвых авиабилетов через Aviasales API.

## Что делает

Бот ведёт диалог: спрашивает откуда, куда, даты и наличие пересадок — и возвращает список билетов с ценами и ссылками на покупку.

## Запуск

1. Создай `.env` в корне проекта:
```
BOT_TOKEN=твой_токен_от_botfather
AVIASALES_TOKEN=твой_токен_от_travelpayouts
```

2. Запусти:
```bash
go run main.go
```
