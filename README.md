# Go Requests

Консольная утилита для поиска дешёвых авиабилетов через API Aviasales.

## Что делает

Запрашивает билеты по заданному маршруту и датам, выводит список рейсов с ценами:

```
Самые дешевые билеты на выбранные даты:
Вылет: 01.05.2026 21:10. Прилет: 01.06.2026 09:25. Дней: 30. Цена: 60143 ₽
```

## Запуск

```bash
go run main.go
```

## Настройка

В `main.go` поменяй параметры запроса: `origin`, `destination`, `departure_at`, `return_at`, `token`.
